# Airline

A fancy project to sell plane tickets.

## Requirements

> Node.js >= 8.x

> firebase-tools [github](https://github.com/firebase/firebase-tools)

## Installation

```
cd functions
npm i 
firebase login
```

## Run local

```
cd functions
node run test.js
```

## Deployment
```
cd functions
npm run deploy
```
